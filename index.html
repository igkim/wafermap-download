2. 📘 Color Wafer Map 설명서 요약

  • 산출물: C# (.NET Framework 4.5) 기반의 Wafer Map 시각화 라이브러리 및 샘플 프로젝트 포함
  • 주요 구성: DLL, 테스트 프로젝트, 디지털 서명 인증서
  • 지원 OS: Windows 8 이상, .NET Framework 4.5 이상
  • 기능: Wafer 및 Die 색상 표현, 사용자 정의 색상 설정, Tooltip 및 Zoom 기능
  • API 예시: `SetBarLegendInfo`, `SetWaferInfo`, `SetDies`, `ZoomIn`, `AddArrowAnnotation` 등
  • 이벤트: `DieClicked` 이벤트를 통한 상호작용
